## trawl

The **session asset extraction** feature — the `sidekick extract` CLI command, the
`Sidekick: Extract Session Assets` VS Code command, and the underlying
`sidekick-shared` extractor APIs (`gatherAssetsForCwd`, `extractUrls`,
`extractFilePaths`, `extractCommands`, `readClaudeAssets`, `readCodexAssets`) —
is adapted from **trawl** by Juan Fourie ([@B33pBeeps](https://github.com/B33pBeeps)),
contributed in [PR #17](https://github.com/cesarandreslopez/sidekick-agent-hub/pull/17).
